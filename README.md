# 🧠 Treinando POO

Este repositório contém uma coleção de **12 exercícios de C#** grande parte de Programção Orientada a Objeto(POO) desenvolvidos como parte de uma avaliação acadêmica.

O objetivo principal foi aplicar na prática os conceitos fundamentais de POO utilizando uma linguagem orientada a objetos.

---

## 📚 Conteúdo abordado

Durante os exercícios, foram trabalhados conceitos como:

- Classes e objetos
- Atributos e métodos
- Encapsulamento
- Construtores
- Herança
- Polimorfismo
- Organização de código orientado a objetos

---

## 🧩 Estrutura do projeto

O repositório está organizado em exercícios individuais, onde cada um aborda um problema específico proposto em aula.
(O exercicio 9 só acompanha a parte final)

## 🎯 Objetivo

Este projeto foi desenvolvido com fins educacionais, com o objetivo de:

- Praticar lógica de programação
- Consolidar conceitos de POO
- Melhorar a organização e estruturação de código
- Preparar para avaliações práticas
